# OMS life cycle

## Order Capture

Use Cases:
- สร้างคำสั่งซื้อใหม่ (Create Order)
- รับ order จากหลายช่องทาง:
    - Website
    - POS
    - Marketplace (Shopee/Lazada/Tiktok)
- B2B Portal
- Bulk Order Import

## Product & Catalog Validation

Use Cases:
- ตรวจสอบ SKU
- Tax Calculation
- Shipping Fee Calculation
- Inventory Availability Check

## Inventory Reservation

Use Cases:
- Reserve stock
- Split reservation
- Multi-warehouse sourcing
- Safety stock protection
- Drop-shipping
- Reallocation if stock shortage

## Order Orchestration

Use Cases:
- Order Routing
- Warehouse Assignment
- Vendor Assignment
- Order Hold / Release
- Compliance checks

## Fulfillment & Shipping

Use Cases:
- Pick / Pack / Ship
- Shipment tracking
- Partial shipment
- Multi-package shipment
- Delivery scheduling
- Last-mile coordination
- Click & Collect
- Store Pickup

## Order Status Tracking

Use Cases:
- Pending
- Confirmed
- Picked
- Packed
- Shipped
- Delivered
- Cancelled
- Returned

## Order Modification

Use Cases:
- Cancel order
- Modify order item
- Add/remove items
- Recalculate pricing

## Returns & Reverse Logistics

Use Cases:
- Return request
- Return pickup
- Refund processing
- Restocking
- Disposal

## Exception Handling

Use Cases:
- Payment failure
- Inventory shortage
- Shipment lost
- Duplicate order
- Manual intervention
- Escalation workflow

## Reporting & Analytics

Use Cases:
- Sales reporting
- Return rates
- Order funnel
- Inventory turnover

## Administrative Use Cases

Use Cases:
- Audit logs
- System monitoring

## Enterprise / Advanced OMS Use Cases

Use Cases:
- Omnichannel order visibility
- Distributed Order Management (DOM)
- Multi-brand management
- Marketplace seller management
- Supply chain resilience
